    Руководитель спецсеминара – к.ф.м.н. Виталий Геннадьевич Казаков

    Информационные системы с богатой функциональностью на уровне сервера приложений и развитым http интерфейсом являются одним из доминирующих типов современного программного обеспечения и главной составляющей мира Web 2.0. Работа студентов на семинаре будет строиться как участие в одном из проектов создания таких информационных ресурсов, систем и комплексов в малых проектных группах.

    В ходе проекта учащиеся на практике освоят работу с HTML, CSS, HTTP, JavaScript + AJAX, PHP, SQL и рядом других технологий, необходимых для работы по созданию подобного ПО.

    Выполняемые студентами проекты будут реальными, предназначенными для практического использования в различных областях образования, науки, производства. Некоторые возможные направления проектов:

    1. Информационно поисковые системы и системы поддержки принятия решений, в том числе научные базы данных с развитыми средствами визуализации и моделирования.
    2. Системы электронного обучения, в том числе системы программированного обучения, электронные учебники, системы создания массовых он-лайн курсов и мультимедиа лекций.
    3. Системы управления виртуальными музеями и научными коллекциями.
    4. Веб системы для социальной коммуникации, социальные сети.

    Каждый из осуществляемых проектов достаточно масштабен для того, чтобы в его рамках могли быть сформулированы в дальнейшем темы для магистерской ВКР, темы аспирантского исследования, диссертации на соискание степени кандидата наук.

    С примерами проектов, которые реализованы при участии студентов под научным руководством преподавателя, возможно ознакомиться в Сети:

    1. Информационная система «Электронная структура атомов». - http://grotrian/nsu.ru
    2. Портал музеев СО РАН. 2005-2014гг. - http://sbras.mmc.nsu.ru
    3. Электронная образовательная среда НГУ. - http://bench.nsu.ru
    4. Музей А.А. Ляпунова. - http://lyapunov.vixpo.nsu.ru
    5. Виртуальный архив по истории отечественной кибернетики и информатики. - http://chistory.nsu.ru
    6. Родная история. Учебник для средней школы - http://mmedia.nsu.ru/history
    7. Древнее искусство Сибири. Учебный археологический Web – музей. - http://museum.mmc.nsu.ru

    Полученные на практикуме навыки помогут освоить профессии веб-дизайнера, html-верстальщика, веб-мастера, CEO специалиста, фронт- и бэк- енд разработчика и будут полезны во многих других специализациях в области ИТ.

    От желающих записаться на семинар требуется готовность к освоению новых технологий, значительному времени самостоятельной работы над проектом, затрачиваемому помимо учебных занятий, склонности к программированию.

    Студентов, желающих записаться на данный семинар, прошу копию заявления в деканат продублировать мне на адрес kazakov@phys.nsu.ru

    В случае вопросов и сомнений можете обращаться за консультацией.
